# 05: Vermilion to Celadon City

## TL;DR - Bot Objectives

```
✓ Route 9 → Battle trainers, catch Pokémon
✓ Route 10 → Get Flash HM (optional)
✓ Rock Tunnel → Navigate dark cave (Flash helps)
✓ Lavender Town → Visit Pokémon Tower (can't complete yet)
✓ Route 8 → Underground Path to Celadon
✓ Celadon City → Biggest city, many shops
✓ Game Corner → Get coins, buy Pokémon (Porygon, Dratini)
✓ Rocket Hideout → Defeat Giovanni, get Silph Scope
✓ Return to Lavender → Complete Pokémon Tower, get Poké Flute
✓ Celadon Gym → Defeat Erika
```

**Time:** ~90-120 minutes  
**Pokédex Progress:** +10-15 species  
**Level Target:** 28-32 for Erika  
**Critical Items:** Flash HM, Silph Scope, Poké Flute

---

## Overview

This is a LONG chapter with multiple major locations. You'll navigate a dark cave, encounter ghosts, infiltrate Team Rocket's hideout, and explore the biggest city in Kanto.

**Key Milestones:**
- Rock Tunnel (first dark dungeon)
- Lavender Town (ghost Pokémon!)
- Celadon City (shopping paradise)
- Rocket Hideout (Team Rocket base)
- Pokémon Tower (rescue Mr. Fuji)
- Fourth gym badge (Rainbow Badge)
- Poké Flute (wake Snorlax)

### Getting Flash HM

**Location:** House south of Pokémon Center (need Cut to access)

**Actions:**
1. Use Cut on tree blocking path
2. Enter house
3. Talk to Professor's Aide
4. If you have 10+ Pokémon in Pokédex, receive **HM05 (Flash)**

**Flash HM:**
- Normal-type move, 70% accuracy
- Lowers opponent's accuracy in battle
- Lights up dark caves outside battle
- **VERY USEFUL for Rock Tunnel**

**Recommended Flash users:**
- Pikachu
- Magnemite
- Voltorb
- Any Pokémon you don't use in battle

**Is Flash required?**
- NO - You can navigate Rock Tunnel without Flash
- But it's MUCH harder (can't see walls)
- Bot should use Flash for easier navigation

**Catching Priority:**
1. **Machop** (25% rate) - Fighting-type, evolves to Machamp (trade evolution)
2. **Onix** (10% rate) - Rock/Ground, big snake

**Items in Rock Tunnel:**
- TM30 (Teleport) - 1F
- Escape Rope - 1F
- Potion - 1F
- TM04 (Whirlwind) - B1F

### Trainers in Rock Tunnel

**Total: 21 trainers**  
**Total Earnings:** ~$2500

**Trainer Types:**
- Picnicker (female trainers)
- Hiker (male trainers)
- Poké Maniac (rare Pokémon trainers)

**Notable Trainers:**
- Poké Maniac: Cubone (Lv23), Slowpoke (Lv23)
- Hiker: Geodude ×3 (Lv21)
- Picnicker: Oddish/Bellsprout (Lv22)

**Strategy:**
- Use Flash to see clearly
- Battle all trainers (huge EXP boost)
- Heal at Pokémon Center if HP low (can exit and return)

### Navigation Tips

**With Flash:**
- Can see walls and paths clearly
- Follow main path north → east → down ladder → south → exit

**Without Flash:**
- Very difficult
- Must memorize path or use map
- Easy to walk into walls
- Not recommended for bot

**Optimal Path:**
1. Enter from Route 10 (north)
2. Go east, battle trainers
3. Find ladder to B1F
4. Navigate B1F south
5. Find ladder back to 1F
6. Exit south to Route 10 (south)

**Time in Rock Tunnel:** ~30-40 minutes with Flash

### Celadon Mansion

**Location:** North of city

**Floors:**
- 1F: Entrance, mailboxes
- 2F: Nothing
- 3F: **Game Freak office!**
  - Talk to developer
  - Receive **Eevee** (gift Pokémon!)
- Rooftop: Nothing

**Eevee:**
- Normal-type
- Can evolve into 3 forms (in Gen 1):
  - Vaporeon (Water Stone)
  - Jolteon (Thunder Stone)
  - Flareon (Fire Stone)

**Recommendation:** Vaporeon or Jolteon (both better than Flareon)

## Game Corner

### Overview
Casino with slot machines. Can win coins and exchange for prizes.

**Location:** Southeast Celadon City

**How it works:**
1. Buy coins at counter ($1000 for 50 coins)
2. Play slot machines
3. Win more coins (or lose them)
4. Exchange coins for prizes at Prize Corner next door

**Slot Machines:**
- Random chance
- Can win 15, 100, or 300 coins
- Very slow to farm coins
- **Not recommended for bot** (too slow)

**Finding Coins:**
- Check floor for hidden coins
- Some people drop coins
- Can find ~20-30 coins total

### Prize Corner

**Pokémon Prizes:**
| Pokémon | Coins | Version | Notes |
|---------|-------|---------|-------|
| Abra | 180 | Both | Psychic-type |
| Clefairy | 500 | Both | Normal/Fairy |
| Nidorina | 1200 | Red only | Poison-type (female) |
| Nidorino | 1200 | Blue only | Poison-type (male) |
| Dratini | 2800 | Red only | **Dragon-type, rare!** |
| Dratini | 4600 | Blue only | **Dragon-type, rare!** |
| Scyther | 5500 | Red only | **Bug/Flying, version exclusive** |
| Pinsir | 2500 | Blue only | **Bug, version exclusive** |
| Porygon | 9999 | Both | **Normal-type, VERY RARE!** |

**TM Prizes:**
| TM | Move | Coins |
|----|------|-------|
| TM23 | Dragon Rage | 3300 |
| TM15 | Hyper Beam | 5500 |

**Critical Pokémon:**
1. **Porygon** (9999 coins) - Only way to get Porygon in Gen 1!

**Bot Strategy:**

- Buy remaining coins with money
- Prioritize: Porygon > Dratini > Scyther/Pinsir

- Moves: Petal Dance, Poison Powder, Mega Drain, Sleep Powder
- Weak to: Fire, Flying, Ice, Psychic
- **Petal Dance is very strong!**
- **Sleep Powder inflicts Sleep!**

### Battle Strategy

**Best Strategy: Use Fire-type**
```
If you have Charizard/Charmeleon:
Turn 1: Ember/Flamethrower on Victreebel (1-2 hit KO)
Turn 2: Ember/Flamethrower on Tangela (1 hit KO)
Turn 3: Ember/Flamethrower on Vileplume (1-2 hit KO)
Easy victory
```

**If you have Flying-type:**
```
Pidgeotto/Pidgeot:
- Wing Attack on all Pokémon
- 2x super effective
- Should win easily
```

**If you DON'T have Fire or Flying:**
```
Use Psychic-type:
- Kadabra: Psybeam/Confusion
- Super effective against Poison-type

Or use Ice-type moves:
- Ice Beam (TM from Department Store rooftop)
- Teach to Water-type Pokémon

Or overlevel and brute force:
- Level 35+ starter
- Use strongest moves
```

**Status Management:**
- Erika's Pokémon use Sleep Powder and Poison Powder
- Bring Awakening (cure Sleep) and Antidote (cure Poison)
- Or use Full Heal (cures all status)

**Recommended Team:**
- Charizard/Charmeleon (Lv30+) - Best choice
- Pidgeotto/Pidgeot (Lv30+) - Good choice
- Kadabra (Lv28+) - Psychic-type
- Any Pokémon with Ice Beam

```

4. Route 8 → Underground Path → Route 7

8. Return to Pokémon Tower → Complete all floors
9. Rival battle on 7F
10. Get Poké Flute from Mr. Fuji
11. Celadon Gym → Defeat Erika
```

### Decision Points

**Flash HM:**
- Get Flash before Rock Tunnel
- Makes navigation much easier
- Teach to HM slave Pokémon

**Eevee Evolution:**
- Choose based on team needs
- Vaporeon: If need Water-type
- Jolteon: If need Electric-type

**Game Corner Priorities:**
